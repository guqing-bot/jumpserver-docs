---
title: "Edit user"
---

The edit user feature is used to modify user information, manage permissions, and control account status, helping administrators easily maintain users and manage system access.

## Edit a user

<Steps>

<Step>
At the top-left of the page, select <Icon icon="arrow-left-right" />, then click <b>Console</b>.

<Frame>
<img src="/images/docs/navs/nav-console.png" alt="image" width="300"/>
</Frame>
</Step>

<Step>
In the left menu, under the USERS section, click **<Icon icon="user" /> Users**.
</Step>

<Step>
In the user table, find the user that you want to edit.
</Step>

<Step>
In the Actions column, click **Edit**.
</Step>

<Step>
Modify user information. About user parameters, see [User parameters reference](/docs/users/user-parameter-reference).
</Step>

<Step>
Click **Submit**.
</Step>

</Steps>

## Bulk edit users

<Steps>

<Step>
At the top-left of the page, select <Icon icon="arrow-left-right" />, then click <b>Console</b>.

<Frame>
<img src="/images/docs/navs/nav-console.png" alt="image" width="300"/>
</Frame>
</Step>

<Step>
In the left menu, click **<Icon icon="user" /> Users**.
</Step>

<Step>
In the user table, check the users you want to edit.
</Step>

<Step>
Above the table, select **Actions <Icon icon="chevron-down" />** button, then click **<Icon icon="edit" /> Edit selected**.
</Step>

</Steps>
