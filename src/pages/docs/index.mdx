# Introduction

## What is JumpServer? 
 
**JumpServer** is an open-source Privileged Access Management (PAM) tool that provides DevOps and IT teams with on-demand and secure access to SSH, RDP, K10s, Remote Apps and Database endpoints through a web browser.


**JumpServer** utilizes a backend technology stack of `Python/Django` and a frontend technology stack of `Vue.js/ElementUI`, in compliance with `Web 2.0` standards.
