## About JumpServer

**JumpServer** is an open-source Privileged Access Management (PAM) tool that provides DevOps and IT teams with on-demand and secure access to SSH, RDP, Kubernetes, Remote Apps and Database endpoints through a web browser.

![jumpserver-overview](/images/jumpserver-overview.png)

JumpServer is composed of multiple core components that work together to form its functional framework, providing users with comprehensive operation management and security control capabilities.
