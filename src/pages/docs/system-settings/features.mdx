# Features

This topic describes about Organiztions settings of JumpServer.

## Configure Announcement
1. In the upper-right corner of any page in JumpServer, click <img src="/images/docs/svg/nav-setting.svg" alt="x" width="15" style={{ display: "inline-block" }}/>.
  <img src="/images/docs/admin-guides/system-settings/nav-settings.png" alt="image" width="450"/>

2. Navigate to **System settings > Features > Announcement**.

3. In the **Announcement** field, enable or disable the Announcement feature.

4. In the **Subject** field, enter a title of the Announcement.

5. In the **Content** field, enter the content of the Announcement. Support MarkDown, HTML and CSS syntax.

6. Optionally, in the **Date Start** and **Data end** fields, set the Announcement validation period.

7. Optionally, in the **More link** field, set the Announcement **View More** external URL.

8. Click **Submit** to confirm.

## Configure Ticket

1. In the upper-right corner of any page in JumpServer, click <img src="/images/docs/svg/nav-setting.svg" alt="x" width="15" style={{ display: "inline-block" }}/>.
  <img src="/images/docs/admin-guides/system-settings/nav-settings.png" alt="image" width="450"/>

2. Navigate to **System settings > Features > Ticket**.

3. In the **Ticket** field, enable or disable Ticket feature.

4. In the **Approval without login** field, enable or disable the ticket approval link login-free feature.

5. In the **Period** and **Unit** field,set the validity period for ticket approval

6. Click **Submit** to confirm.

## Configure Account storage
1. In the upper-right corner of any page in JumpServer, click <img src="/images/docs/svg/nav-setting.svg" alt="x" width="15" style={{ display: "inline-block" }}/>.
  <img src="/images/docs/admin-guides/system-settings/nav-settings.png" alt="image" width="450"/>

2. Navigate to **System settings > Features > Account storage**.

3. In the **Vault** field, enable or disable Vault feature.

4. In the **Record limit** field, set the record limit. Basic Vault Record limit 999 If the specific value is less than 999 (default), the system will automatically perform a task every night: check and delete historical accounts that exceed the predetermined number. if the value reaches or exceeds 999 (default), no historical account deletion will be performed

5. Click **Submit** to confirm.

## Configure Chat AI 

1. In the upper-right corner of any page in JumpServer, click <img src="/images/docs/svg/nav-setting.svg" alt="x" width="15" style={{ display: "inline-block" }}/>.
  <img src="/images/docs/admin-guides/system-settings/nav-settings.png" alt="image" width="450"/>

2. Navigate to **System settings > Features > Chat AI**.

3. In the **Chat AI** field, enable or disable Chat AI feature.

4. In the **Base URL** field, enter API Base URL .

5. Optionally, in the **Proxy** field, enter AI proxy URL.

6. In the **DeepSeek model** field, select model type.

7. Click **Submit** to confirm.
