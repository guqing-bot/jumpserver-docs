# General settings

This topic describes about General settings of JumpServer.

## Site URL

### How to configure

<Steps>

<div>
<Nav name="system-settings" />
</div>

<div>
Navigate to **System settings > General**.
</div>

<div>
In **Site URL** fields, enter an accessible address for current JumpServer service. The **Site URL** is utilized to generate HTTP/HTTPS links within system messages.
</div>

<div>
Click **Submit**.
</div>

</Steps>

### Usage scenarios

Site URL is the externally accessible address of the current product service and isusually used in links in system emails


## Document URL

### How to configure

<Steps>

<div>
<Nav name="system-settings" />
</div>

<div>
Navigate to **System settings > General**.
</div>

<div>
In **Document URL** fields, enter an accessible address for product document site(default official document site). Document URL refers to the address in the top navigation bar **Help > Document**.
</div>

<div>
Click **Submit**.
</div>

</Steps>

### Usage scenarios

When users click <Icon name="help"/> located in the upper-right corner of any page in JumpServer, they will be redirected to this URL after clicking "Docs".
