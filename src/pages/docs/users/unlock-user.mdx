# Unlock user

<Alert>
When a user fails to log in many times within a short period, the system temporarily locks the user account to enhance security and prevent brute-force attacks.
</Alert>

<Steps>

<div>
<Nav name="console" />
</div>

<div>
In the table, click the name of the user you want to unlock.
</div>

<div>
On the user details page, click **Basic** tab.
</div>


<div>
In the **Quick update** card, find the "Unlock user" section and click the **Unlock** button.
</div>

</Steps>
