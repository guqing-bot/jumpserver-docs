---
title: "Reset user MFA"
---

When a user cannot log in due to a lost MFA, the administrator can reset MFA for the user.

<Steps>

<Step>
At the top-left of the page, select <Icon icon="bars" />, then click <Icon icon="desktop" /> **Console**.
</Step>

<Step>
In the left menu, click **<Icon icon="user" /> Users**.
</Step>

<Step>
In the user table, click the name of the user you want to reset.
</Step>

<Step>
On the user details page, click **Basic** tab.
</Step>

<Step>
In the Quick update card, find the "Reset mfa" section and click **Reset**.
</Step>

</Steps>
