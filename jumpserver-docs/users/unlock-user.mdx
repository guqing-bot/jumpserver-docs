---
title: "Unlock user"
---

When a user fails to log in many times within a short period, the system temporarily locks the user to enhance security and prevent brute-force attacks. The administrator can unlock the user.

<Steps>

<Step>
At the top-left of the page, select <Icon icon="bars" />, then click <Icon icon="desktop" /> **Console**.
</Step>

<Step>
In the left menu, click **<Icon icon="user" /> Users**.
</Step>

<Step>
In the user table, click the name of the user you want to unlock.
</Step>

<Step>
On the user details page, click **Basic** tab.
</Step>

<Step>
In the Quick update card, find the "Unlock user" section and click **Unlock**.
</Step>

</Steps>
